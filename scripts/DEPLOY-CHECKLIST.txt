Backend
[ ] dotnet build en backend/GestionSpa.Api

Frontend (Cloudflare Pages)

[ ] Build: npm ci && npm run build
[ ] Output: dist
[ ] VITE_API_URL=https://gestionspa-production.up.railway.app/api  (debe terminar en /api)

Railway API

[ ] CORS_ALLOW_PAGES_DEV=false  (prod real)
[ ] ASPNETCORE_ENVIRONMENT=Production
[ ] PORT=8080 (default en Program.cs)
